# Pong Game (Python)

This is a simple implementation of the classic **Pong** game built using **Python**.  
The project demonstrates basic game development concepts such as movement, collision detection, and score tracking.

## Features
- Two-player paddle control
- Ball movement and collision logic
- Score system
- Simple and clean gameplay

## Technologies Used

- Turtle / Pygame *(qaysi birini ishlatgan bo‘lsang qoldir, bittasini o‘chir)*
